# 🍛 Random Tamsang (สุ่มอาหารตามสั่ง)

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev/)
[![TailwindCSS](https://img.shields.io/badge/tailwindcss-%2338B2AC.svg?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com/)
[![DaisyUI](https://img.shields.io/badge/daisyui-58337E?style=for-the-badge&logo=daisyui&logoColor=white)](https://daisyui.com/)

An interactive web application to help you decide what to eat for your next "Tamsang" (Thai made-to-order) meal. No more "Gin Arai Dee?" (What should I eat?) dilemmas!

## ✨ Features

- **Random Selection**: Instantly pick a random dish based on your preference.
- **Spice Level Filter**: Choose between Spicy and Non-Spicy menus.
- **Menu Gallery**: Browse through all available delicious Thai dishes.
- **Modern UI**: Clean, responsive design using Tailwind CSS and DaisyUI.
- **Interactive Feedback**: SweetAlert2 for beautiful alerts and modals.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS & DaisyUI
- **Routing**: React Router DOM
- **Icons**: Font Awesome
- **Popups**: SweetAlert2

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/randomTamsang.git
   ```

2. Navigate to the project directory:
   ```bash
   cd randomTamsang
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

```text
src/
├── assets/          # Static assets like images and logos
├── components/      # Reusable UI components (Navbar, Footer)
├── App.jsx          # Main randomizer logic and UI
├── Menu.jsx         # Menu list view
├── Spicy.jsx        # Data for spicy menus
└── NotSpicy.jsx     # Data for non-spicy menus
```
